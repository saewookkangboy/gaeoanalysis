## 1. Admin 구독 관리 기능 테스트

### 1.1 Admin 구독 관리 페이지 접근
- [ ] `/admin/subscriptions` 페이지 접근 가능
- [ ] 구독 목록이 정상적으로 표시되는지 확인
- [ ] 필터링 기능 (플랜, 상태, 검색) 정상 동작 확인

### 1.2 Admin 구독 조회 API
- [ ] `GET /api/admin/subscriptions` 정상 응답
- [ ] 필터 파라미터 정상 동작 확인
- [ ] 사용량 정보 포함 여부 확인

### 1.3 Admin 구독 변경 API
- [ ] `PATCH /api/admin/subscriptions/[subscriptionId]` 정상 동작
- [ ] 플랜 변경 정상 동작 확인
- [ ] 권한 검증 확인

## 2. 소셜 로그인 후 분석 프로세스 테스트

### 2.1 비로그인 상태에서 분석 시작
- [ ] 비로그인 상태에서 "분석 시작" 버튼 클릭
- [ ] 로그인 모달이 정상적으로 표시되는지 확인
- [ ] 입력한 URL이 모달에 표시되는지 확인

### 2.2 로그인 후 자동 분석 시작
- [ ] 소셜 로그인 완료 후 자동으로 분석 페이지로 리디렉션
- [ ] 입력한 URL이 정상적으로 복원되는지 확인
- [ ] 분석이 자동으로 시작되는지 확인

### 2.3 에러 처리
- [ ] 로그인 실패 시 적절한 에러 메시지 표시
- [ ] 분석 실패 시 적절한 에러 메시지 표시

## 3. AI Visibility + AIO 알고리즘 테스트

### 3.1 AI Visibility 점수 계산
- [ ] 분석 결과에 `aiVisibilityScore` 포함 여부 확인
- [ ] AI Visibility 점수가 0-100 범위인지 확인
- [ ] AI Visibility 추천사항 포함 여부 확인

### 3.2 AIO 점수 계산
- [ ] ChatGPT, Perplexity, Gemini, Claude 점수 포함 여부 확인
- [ ] 각 점수가 0-100 범위인지 확인
- [ ] AIO 분석 결과 포함 여부 확인

### 3.3 데이터베이스 저장
- [ ] 분석 결과가 DB에 정상적으로 저장되는지 확인
- [ ] `ai_visibility_score` 필드 저장 확인
- [ ] `ai_visibility_recommendations` 필드 저장 확인
